# SoYummy Frontend

Frontend part of the SoYummy recipe application.

SoYummy is a recipe app where users can search recipes, view recipe details, add recipes to favorites, create their own recipes, manage a shopping list, and update their profile.

## Tech Stack

- React
- React Router
- Redux Toolkit
- Redux Persist
- Axios
- Formik
- Yup
- Styled Components
- Emotion
- Material UI
- React Select
- React Toastify
- React Hot Toast
- React Icons

## Features

- User registration and login
- Responsive layout
- Recipe search
- Recipe details page
- Recipes by categories
- Popular recipes
- Favorite recipes
- Add own recipes
- Shopping list
- User profile update
- Form validation
- Notifications and loaders
- Integration with backend API
